# Projet-Python
Dépôt GitHub pour le projet Python pour la data science.

Projet réalisé par Alix HÉRITIER et Ivan TISSOT.

1. Objectif

L'objectif de ce projet est d'explorer la richesse des données électorales de 2022 et des données CSP (Source : INSEE) avec Python et de construire un dépôt GitHub permettant la reproductibilité du code.
Nous réaliserons une combinaison de plusieurs bases de données (électorales et CSP) puis nous ferons une ACP avec des K-means. Nous ferons ensuite de la cartographie avec le package 'cartiflette' en utilisant les résultats obtenus avec l'ACP + K-means. Nous analyserons ensuite les clusters obtenus avec l'ACP + K-means. 

2. Présentation du dépôt

Notre production est contenue dans un fichier notebook Jupyter : main.ipynb

Le dossier scripts contient des fichiers Python (.py) contenant chacun une fonction qui est utilisé dans le notebook. Cela permet d'améliorer la lisibilité de notre code.

Le fichier requirements.txt permet d'installer l'ensemble des librairies nécessaires pour que le code s'éxecute sans engendrer d'erreur.
